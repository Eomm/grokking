# Tree

The tree is a great data-structure that fit in very broad situations based on how much:

- searches
- addition
- deletions

you will need to do.


**Comparison**

| Tree          | Children | Ordered     | Search   | Insert   | Remove   |
|---------------|---------:|-------------|----------|----------|----------|
|Red-Black Tree |        2 | Ordered     | O(log n) | O(log n) | O(log n) |
